------------------------------------------------------------------------
Heating Boiler Example
------------------------------------------------------------------------
The IEC 61499 Heating Boiler Example has been implemented based on the IEC 61131 Demo Boiler example provided by PLCopen [http://plcopen.org/] and OPC Foundation [http://www.opcfoundation.org].

------------------------------------------------------------------------
How to use Tutorial
------------------------------------------------------------------------
1. copy project into workspace and import it into 4DIAC-IDE <File/Import/General/Existing Projects into Workspace>
2. export Function Blocks to 4DIAC-RTE <File/Export/Distributed Control System/FORTE Type Export>, mark <tutorials/boiler>, select 4DIAC-RTE directory <xxx/forte\src\modules\tutorials\boiler>
3. add proper CMakeLists.txt files
4. compile 4DIAC-RTE 
5. launch 4DIAC-RTE and FBRT
6. download applications
